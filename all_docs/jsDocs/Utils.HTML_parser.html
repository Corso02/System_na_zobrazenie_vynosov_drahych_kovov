(async, static) parseAll()

Function to parse pages in given range. This function utilizes single for loop to get URL parameter for a year. Firstly it calls Utils.HTML_parser.parsePage, when parsing was successful then Utils.HTML_parser.writeTableToFile is called to write parsed table to file.

Function to parse page with given URL address. Retrieves HTML content of page by calling Utils.HTML_parser.getPage. Then loads html using Cheerio, and thanks to that parses table from given page. To parse different page, you need to update selector for table to be parsed on line 43.
