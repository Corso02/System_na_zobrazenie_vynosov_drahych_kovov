(async, static) validateForm()

Function to validate form and generate new table. At first this function checks type of table user wants to generate based on input with "table_type" id. If selected type is for investment calculator then Validation.validateFormForCalc function is called. Otherwise it calls Validation.check_years_input function, if everything is alright then API request is send to retrieve data for table. Then new instance of Tables.TableGenerator class is created and Tables.TableGenerator.generateTable is called. If there already is a generated table user is prompted if he wants to add new table or overwrite existing one. Based on response Tables.addBasicTable or Tables.overwriteTable is called. If this is first table to be added than Tables.addTable is called.
